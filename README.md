# 🧠 EEG Signal Anomaly Detection using Deep Learning

## 📌 Description
This project analyzes EEG (Electroencephalogram) signals using deep learning models to detect anomalies such as seizures.

It uses ConvLSTM Autoencoder and Spectral SE-ResNet to reconstruct EEG signals and identify anomalies based on reconstruction error.

---

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib

---

## 🔍 Features
- EEG signal preprocessing
- Deep learning-based reconstruction
- Anomaly detection using reconstruction error
- Visualization of results
